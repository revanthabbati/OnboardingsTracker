Before connecting:
  1. Open your Google Sheet
  2. Click Share → General access → set to "Anyone with the link" (Viewer)
  3. Copy the URL and paste it above
This app reads data client-side via Google's public visualization endpoint — no API key needed, and your sheet is never modified. Note: rich-text hyperlinks inside cells (e.g. Checklist / Sales Hand Off links) export as text only; the underlying link target isn't accessible from the sheet's public export.
